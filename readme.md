# Custom Browser Homepage

A lightweight and fully customizable start page for web browsers.
Designed to create a personal new-tab experience with quick links, custom wallpapers, and flexible layout controls.

## Features

* Clean and minimal layout
* Custom quick links
  * Add, edit, reorder, and remove websites
  * Import and export links as JSON
* Background Gallery

  * Supports static images and animated wallpapers
  * Add, remove, and reorder backgrounds
  * Simple interface to manage your gallery
* Local configuration

  * All settings are stored locally in the browser (no backend required)
* Column customization

  * Adjust the number of columns in the links grid
* Lightweight, responsive, and fast

## Installation

1. Download the project and load it manually as an unpacked extension in your browser
   or
2. Install it directly from the Chrome Web Store:
   [https://chromewebstore.google.com/detail/custom-browser-homepage/egokchnpmiknnlabdbndnnallmnbmpfg](https://chromewebstore.google.com/detail/custom-browser-homepage/egokchnpmiknnlabdbndnnallmnbmpfg)
