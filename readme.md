# Custom Browser Homepage

A lightweight, customizable start page for web browsers. Provides quick links to personalize the new-tab or homepage experience.

## Features
- Clean, minimal layout
- Customizable links and wallpapers
- Local settings stored in JSON (no backend required)
- Responsive and fast

## Installation
- Download the extension and put in your browser manually or download it from the [Chrome Web Store](https://chromewebstore.google.com/detail/custom-browser-homepage/egokchnpmiknnlabdbndnnallmnbmpfg)
