# Custom Browser Homepage

A lightweight, customizable start page for web browsers. Provides quick links to personalize the new-tab or homepage experience.

## Features
- Clean, minimal layout
- Customizable links and search engines
- Local settings stored in JSON (no backend required)
- Responsive and fast

## Installation
1. Clone the repository:
    git clone https://github.com/your-username/Custom-Browser-Homepage.git
2. Open the project folder in a browser or a local server:
    - Directly open `index.html`, or
    - Serve with a simple HTTP server (e.g., `npx http-server`)

## Usage
- Edit `config.json` (or the provided settings file) to add or modify bookmarks, search providers, and appearance options.
- Open `index.html` in the browser or set it as the browser homepage/new-tab page.

## Customization
- Modify CSS variables in `styles.css` to change colors and spacing.
- Add/remove widgets by editing `index.html` and corresponding scripts.
- Use localStorage to persist user preferences across sessions.
